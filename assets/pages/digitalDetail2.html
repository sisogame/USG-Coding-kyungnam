[공통 Track] 데이터 문해력

교육과정 특징

  • · 데이터를 분석할 때 필요한 다양한 방법론을 학습합니다.
  • · 데이터에 잠재된 패턴을 탐색하는 방법을 이해합니다.
  • · 데이터를 올바르게 해석하고 결론을 도출하는 방법을 이해합니다.

교육과정 로드맵

  • 1단계
    빅데이터의 출현 및 분석 기술

  • · ‘기업이 빅데이터를 한다’는 의미의 이해

    · 빅데이터 활용 분야, 국내 데이터 직무의 향후 5년간 필요 인력, 데이터 직무 구분, 데이터 직무 채용 시 우대기술의 확인

    · 데이터 분석 프로세스, 빅데이터 분석 기술 및 학습 로드맵 확인 (웹크롤링, 데이터 전처리, 데이터 시각화, 탐색적 데이터 분석, 통계 분석, 머신러닝/딥러닝, 자연어 처리, 텍스트 마이닝)

  • 2단계
    디지털 전환 (DX)

  • · 사례를 통한 디지털 전환 과정에 대한 이해

    · 디지털 트윈의 등장과 유망 기술로의 대두

  • 3단계
    데이터 문해력 및 실습

  • · 데이터 문해력에 대한 이해와 부족할 경우에 대한 대비 (데이터 수집, 데이터 해석, 분석 방법 선택)

    · 데이터 문해력 증진 방안의 확인
